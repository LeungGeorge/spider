IntelliJ IDEA 在业界被公认为最好的 Java 开发平台之一，在智能代码助手、代码自动提示、J2EE 支持、Ant、JUnit、代码审查等方面都有不错的表现，且支持基于 Android 平台的程序开发。答案中已经有非常全面的学习资料了，在此推荐 6 款国产 IntelliJ IDEA 优秀插件，好的工匠需要用好的工具才能创造出优秀的艺术品，供大家交流讨论。


1. 项目名称：码云 IntelliJ IDEA 插件 Intellij-Gitee

项目简介： 码云 IntelliJ IDEA 插件由码云推出的 Git 插件，该插件基于 IntelliJ IDEA Github 插件开发，可实现 IntelliJ IDEA 与开源中国码云平台对接，其主要功能如下：

  • 登录功能。输入码云帐号和密码可以直接登录到码云。
  • 获取用户项目列表。在登录码云之后，用户可以获取码云线上项目列表，并可选择该用户的任意项目进行 clone。
  • 推送项目。在登录码云之后，用户可以将项目 push 到码云。
  • 创建项目。对于仅存在于本地的项目，在登录码云后，用户可以直接通过插件在码云上创建、提交项目，不需要在 web 进行操作。

项目地址：开源中国/intellij-gitosc - 码云 Gitee.com


2. 项目名称：IDEA 的 Lua 编辑调试插件 EmmyLua

项目简介： EmmyLua 是一款基于 IntelliJ IDEA 的 Lua 插件，拥有强大的代码提示以及调试功能，开源免费。

项目地址：阿唐/IntelliJ-EmmyLua - 码云 Gitee.com


3. 项目名称：深度集成 xmake 和 Intellij-IDEA 的插件 xmake-idea

项目简介： xmake-idea 插件深度集成了 xmake 和 Intellij-IDEA (CLion/Android Studio/ ..)，提供方便快速的跨平台 c/c++ 构建。

项目地址：tboox/xmake-idea - 码云 Gitee.com


4. 项目名称：Intellij IDEA 翻译插件 intellij-translation

项目简介： Intellij IDEA 系列编辑器的翻译插件, 目前仅提供 英文 -> 中文 翻译。

项目地址：zyuyou/intellij-translation - 码云 Gitee.com


5. 项目名称：Intellij IDEA 代码生成器插件

项目简介： 根据 Java 实体类生成建表语句的 Intellij 插件，帮你完成无趣的代码翻译。

项目地址：票牛/pngen - 码云 Gitee.com


6. 项目名称：IDEA mybatisSQL 提取插件

项目简介： 此功能从 mybatipse 移植过来的，在 idea 插件中未见过此类插件，故移植了该功能。

项目地址：聂秋荣/mybatis-log - 码云 Gitee.com
